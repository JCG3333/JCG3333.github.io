# 🚀 주요 프로젝트

## 1. 업비트 자동매매 시스템 (Upbit Auto Trader)
- **설명:** 업비트 API 기반 실시간 자동매매 봇으로, 변동성 돌파 및 ML 기반 예측 전략을 병행 적용.
- **성과:** 월평균 수익률 5~8%, Discord 알림 및 Google Sheets 기록 자동화.
- **기술 스택:** Python, Upbit API, Google Sheets API, Discord Webhook
- **링크:** [프로젝트 설명 보기](https://github.com/JCG3333/auto-trading-bot)
